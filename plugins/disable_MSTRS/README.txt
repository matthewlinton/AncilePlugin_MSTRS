Disable Microsoft Telemetry Reporting Service

ABOUT
Modify the registry to disable Micrsoft's Telemetry Data Reporting Service.

	
CONFIGURATION
The following options can be added to config.ini

	DISABLEMSTRS (Boolean) - Enable or disable the disable_MSTRS plugin
		Y	- Enable the disable_MSTRS Plugin (DEFAULT).
		N	- Disable the disable_MSTRS Plugin.


NOTE
Disabling this plugin does not undo the changes made in previous runs.
